# Entorno y Configuración

## Variables de Entorno
- `BSC_RPC_URL`: URLs RPC separadas por comas.
- `SOL_RPC_URL`: Endpoint RPC de Solana.
- `BSC_PRIVATE_KEY_1`, `BSC_PRIVATE_KEY_2`, etc.
- `SOL_PRIVATE_KEY_1`, `SOL_PRIVATE_KEY_2`, etc.
- `JITO_RPC_URL`: (Opcional) URL de Jito Block Engine.

## Uso del Binario
La habilidad asume que el binario `solana_claw_coin_cli` está compilado y disponible en la raíz del proyecto o añadido al PATH.
Para compilar: `cargo build --release`
Ubicación del binario: `target/release/solana_claw_coin_cli`
