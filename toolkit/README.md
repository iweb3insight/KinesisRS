| 工具 | 说明 |
|------|------|

## 快速使用

```bash
# 安装
curl -sL https://raw.githubusercontent.com/iweb3insight/KinesisRS/main/scripts/install.sh | bash

# 使用
kinesis-rs --json quote <TOKEN> <AMOUNT> --chain solana
```
